# Projeto CRUD com Django

Projeto desenvolvido para a **Atividade Avaliativa 3** da disciplina de Desenvolvimento Web, utilizando o framework **Django**.

## Descrição do Projeto
O sistema consiste em um **CRUD de Funcionários**, permitindo:
- Cadastro de funcionários
- Listagem de funcionários
- Edição de funcionários
- Exclusão de funcionários

Os dados são armazenados em um **banco de dados MySQL**, garantindo persistência das informações.

---

## Tecnologias Utilizadas
- Python 3
- Django
- MySQL
- HTML
- CSS
- Git e GitHub
